# 博客功能演示

这篇文章演示新增的现代博客能力：自动目录、阅读时长、代码复制与**在线运行**、暗色模式、全文搜索、相关文章等。

## 在线运行 JavaScript

JavaScript 直接在浏览器本地沙箱运行，离线/办公网均可用。点击代码块右上角「运行」：

```js
const nums = [1, 2, 3, 4, 5];
const sum = nums.reduce((a, b) => a + b, 0);
console.log('数组求和 =', sum);
console.log('平方:', nums.map(n => n * n));
```

## 在线运行 Python

Python 通过在线执行服务运行（需要可访问公网）：

```python
def fib(n):
    a, b = 0, 1
    out = []
    for _ in range(n):
        out.append(a)
        a, b = b, a + b
    return out

print("斐波那契:", fib(10))
```

## 在线运行 Java

Java 同样通过在线执行服务编译运行：

```java
public class Main {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 1; i <= 100; i++) sum += i;
        System.out.println("1..100 求和 = " + sum);
    }
}
```

## 写作体验

- 标题超过两个会自动生成**目录**并高亮当前阅读位置
- 文章顶部显示**字数**与**预计阅读时间**
- 顶部细条是**阅读进度**
- 按 `/` 唤起**全文搜索**，支持标题、摘要与正文
- 右上角可切换**暗色模式**
- 文章底部提供**上一篇 / 下一篇**与**相关文章**

> Markdown 即网站：你在发布台写下的内容，编译后即时同步到前台。
